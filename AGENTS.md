# 项目说明

本目录是 PyQt5 的项目集合，其中 Library 下面是一些 PyQt 的库，而 Apps 目录下面是一堆 PyQt 的小项目，每个项目一个文件夹。



## 技术规格

使用 Python 3.8 + PyQt5，方便我打包和在 Windows 7 上使用，要注意 DPI 缩放问题，要感知桌面 DPI 缩放比例，并进行运行时调整，不要在高分屏上看着太小辣眼睛。

每个 Apps 下面的子项目，包含一个 prd.md 和 spec.md 两个文件，分别是产品说明和技术规格，迭代对应子项目需要先阅读对应的两个文档，做出修改后也需要同步回去。
